# Crimes Against Women in India Dashboard

## Overview
An interactive Power BI dashboard analyzing crimes against women across Indian states and union territories.

## Tools Used
- Power BI
- Excel
- Data Cleaning
- Data Visualization

## Features
- State-wise crime analysis
- Crime trend analysis
- Category-wise crime distribution
- Conviction rate comparison
- Interactive dashboards
