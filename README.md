# Crimes Against Women in India Dashboard

## 📌 Project Overview

This project is an interactive Power BI dashboard developed to analyze crimes against women in India using historical crime data. The dashboard helps users understand crime trends, compare states, analyze conviction rates, and identify major crime categories through visual analytics.

The goal of this project is to transform raw crime data into meaningful insights that can support awareness, research, and data-driven decision-making.

---

## 🎯 Objectives

The main objectives of this project are:

- Analyze crimes against women across different states and union territories.
- Identify trends in crime rates over time.
- Compare reported cases and convicted cases.
- Examine conviction rates across regions.
- Study crime categories and their distribution.
- Present insights using interactive visualizations.

## 📊 Dashboard Pages

### 1. India Crime Overview

This page provides a high-level summary of crime statistics across India.

Features:
- Total reported cases
- Total convicted cases
- Conviction rate
- State-wise crime distribution
- Year-wise crime trends
- Key performance indicators (KPIs)

---

### 2. State Analysis

This section focuses on state-level insights.

Features:
- State-wise ranking
- Crime comparison among states
- Reported vs convicted cases
- Conviction rate analysis
- Identification of high-crime regions

---

### 3. Crime Category Analysis

This page analyzes different categories of crimes against women.

Features:
- Category-wise crime distribution
- Most common crime categories
- Category growth trends
- Conviction comparison by category

---

### 4. Time-Based Analysis

This page focuses on historical trends.

Features:
- Year-wise crime growth
- Peak crime years
- Moving trend analysis
- Long-term pattern identification

---

## 📈 Key Insights

Some major findings from the analysis include:

- Certain states contribute significantly to the total number of reported cases.
- Crime cases have generally increased over the years.
- Conviction rates remain lower compared to reported incidents.
- Some crime categories show higher growth trends than others.
- Geographic patterns reveal regional differences in crime statistics.
